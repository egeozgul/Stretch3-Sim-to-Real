# Stretch3-Sim-to-Real

This repo will contain the documentation and sourcecode for Stretch3 sim to real navigation and manipulation tasks

## Lidar Slam Snapshot

Black pixels represent the pre-constructed map of the enviornment using Lidar.
Red points are the realtime lidar measurements that are used for localization via Lidar SLAM.
Table figure is added to show the desk and docking position A which has pose of (x: 0.53 y: 0.49 yaw:90).
